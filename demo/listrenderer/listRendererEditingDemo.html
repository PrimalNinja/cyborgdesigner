Inline Editing Demo - Side by Side Comparison

Excel-Like Editing

  • Click any cell or navigate with keyboard to start editing
  • Tab / Shift+Tab - Move forward/backward through cells
  • Up/Down Arrows - Move up/down same column
  • Enter - Move down (same as Down Arrow)
  • Home / End - Jump to top-left / bottom-right
  • Escape - Cancel edit

WITH Add New Row

allowaddnew = true
  • ✓ Gray blank row at bottom with * selector
  • ✓ Tab/Down Arrow to blank row and start typing
  • ✓ New green rows created automatically
  • ✓ Blank row stays at bottom for next record

WITHOUT Add New Row

allowaddnew = false
  • ✗ No blank row at bottom
  • ✗ No * selector
  • ✗ Tab/Down stops at last data row
  • ✓ Can only edit existing rows (turn yellow)

Edit Only (No Add/Delete)

allowedit = true, allowaddnew = false, allowdelete = false
  • ✓ Can edit existing rows (turn yellow)
  • ✗ No blank row at bottom
  • ✗ No delete functionality
  • ✓ Tab/Down stays at last cell

Read-Only Override

readonly = true (overrides allowedit/allowaddnew/allowdelete)
  • ✓ Editable grid config with readonly override
  • ✓ Has row selector column (from editable config)
  • ✓ EXPAND mode (no pagination, like editable)
  • ✗ No search toolbar
  • ✗ No editing allowed
  • ✓ Click row to select

Callbacks Reject All Changes

All 3 callbacks return false
  • ✗ onRowUpdate returns false → No yellow status
  • ✗ onRowAddNew returns false → Can't add rows
  • ✗ onRowDelete returns false → Can't delete rows
  • ✓ All changes are rejected (validation demo)
