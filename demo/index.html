01 — List Renderer
List Renderer — Display
Data Views

Renders record sets in multiple presentation modes from a single data source. Supports List (tabular) and Gallery (thumbnail grid) views, each available in three flow strategies: paged navigation, infinite/continuous scroll, and full-expand. Columns are sortable (single and multi-column with Shift+Click) and the table includes live search.

List Renderer — Inline Editing
Editing Demo

Extends the List Renderer with Excel-like inline cell editing. Click any cell or navigate with keyboard shortcuts — Tab, Shift+Tab, arrow keys, Home, End, and Enter — to move through the grid. Supports optional "Add New Row" functionality. A side-by-side comparison demo shows the difference between grids with and without add-new enabled.

02 — Form Designer
Cyborg Designer
Visual IDE

A drag-and-drop layout designer that outputs portable JSON definitions. Design every aspect of a form — fields, sections, validation, layout, and data types — through a tabbed visual interface covering Application, Dashboard, Database Schema, Form, Menu, Report, and AI Orchestration. Supports unlimited field types: the architecture is metadata-driven with callbacks, so you define your own types without touching the designer core. The output JSON drives both the Form Renderer and the List Renderer.

03 — Form Renderer
Form Renderer
Runtime Engine

Takes a JSON form definition produced by Cyborg Designer and renders it as a fully interactive HTML form at runtime — no code changes needed. Ships with a comprehensive set of built-in field types (textboxes, date pickers, GPS capture, barcode scanners, rich HTML editors, code editors, charts, file uploads, and more), and is designed to be extended — register your own renderers for custom field types to match whatever the designer sends down. Provides built-in validation, data population, and a clean API for reading form values back into your application.
